perfil de egreso

la formacion que ejerce la carrera de tecnico programador permite al egresado, atraves de la articulacion de saberes de diversos campos, realizar actividades dirigidas a la:

  • instalacion y desarrollo de software de aplicacion utilizando programacion estructurada y orientada a objetos en ambientes web y moviles, con almacenamiento de datos,
  • asi como la configuracion y administrador de plataforma e-learning y
  • comercio electronico
durante el proseso de formacion de los 5 modulos, el estudiante desarrollara o reforzara:
las siguientes competencias profecionales:
  • desarrolla e instala software de aplicacion utilizando programacion estructurada, con almacenamiento persistente de datos
  • desarrolla software de aplicacion utilizando programacion orientada a objetos, con almacenamiento persistente de datos
  • deesarrolla aplicaciones web y mobiles
  • administra sistemas operativos y servicios
  • desrrolla, administra y configura soluciones de e-learning y comercio eletronico
